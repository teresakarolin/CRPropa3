Detailed Description

All information about the cosmic ray.

The Candidate is a passive object, that holds the information about the state of the cosmic ray and the simulation itself.

Add a new candidate to the list of secondaries.

Parameters
idparticle ID of the secondary
energyenergy of the secondary

Adds a new candidate to the list of secondaries of this candidate. The secondaries Candidate::source and Candidate::previous state are set to the source and previous state of its parent. The secondaries Candidate::created and Candidate::current state are set to the current state its parent, except for the secondaries current energy and particle id. Trajectory length and redshift are copied from the parent.
